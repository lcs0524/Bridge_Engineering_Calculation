## 项目初始化提示词

### 阶段1：项目初始化
```
作为前端架构师，请创建一个Vue3 + TypeScript项目用于桥梁工程安全评估软件。

技术要求：
- Vue 3.3+ 组合式API
- TypeScript 5.0+
- Vite 4.0+ 构建工具
- Element Plus UI组件库
- Tailwind CSS 样式框架
- ECharts 5.0+ 图表库
- Pinia 状态管理
- Vue Router 4 路由
- Electron 桌面应用打包

项目结构：
src/
├── api/           # API接口
├── components/    # 组件库
├── stores/        # 状态管理
├── views/         # 页面组件
├── utils/         # 工具函数
└── types/         # TypeScript类型

功能需求：
- 桥梁沉降分析

- 电线塔基础分析
- 结果可视化
- 报告导出

请创建完整的项目脚手架，包括所有配置文件和基础目录结构。
```

## 使用说明

### 快速开始
1. 按顺序使用阶段1-5的提示词
2. 每个阶段完成后验证功能
3. 使用专用提示词进行细节优化
4. 最后进行测试和打包

### 注意事项
- 保持现有Python后端不变
- 逐步迁移，可并行运行两套系统
- 充分测试所有功能
- 收集用户反馈持续优化
